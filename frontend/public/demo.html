📚 Anleitung zur Verwendung

1

🚀 App starten

Klicken Sie auf "→ jetzt testen ←" und melden Sie sich mit den Demo-Anmeldedaten an:

  • Benutzer: demo / demo123
  • Admin: admin / admin123
2

📝 Erste Liste erstellen

Nach der Anmeldung:

  • Klicken Sie auf "+ Neue Liste"
  • Geben Sie einen Namen ein (z.B. "Wocheneinkauf")
  • Bestätigen Sie mit Enter oder dem ✓ Button
3

🛒 Artikel hinzufügen

In Ihrer Liste:

  • Tippen Sie Artikel in das Eingabefeld (z.B. "Milch")
  • Wählen Sie eine Kategorie aus dem Dropdown
  • Nutzen Sie ⭐ für Favoriten
  • Artikel werden automatisch gruppiert
4

🎯 Erweiterte Funktionen

Entdecken Sie weitere Features:

  • Favoriten: Häufige Artikel schnell hinzufügen
  • Export: Liste als Text herunterladen
  • Admin-Panel: Artikel und Kategorien verwalten
  • Bulk-Aktionen: Mehrere Artikel gleichzeitig bearbeiten

❓ Häufig gestellte Fragen (FAQ)

🔐 Sind meine Daten sicher in der Demo?

+

Ja! Die Demo läuft vollständig offline in Ihrem Browser. Alle Daten werden nur lokal gespeichert und niemals an externe Server gesendet. Ihre Privatsphäre ist geschützt.

💾 Bleiben meine Demo-Daten erhalten?

+

Ja, Ihre Listen und Artikel werden im Browser localStorage gespeichert und bleiben auch nach dem Schließen des Browsers erhalten. Sie können jederzeit zurückkehren und weiterarbeiten.

🌐 Funktioniert die Demo ohne Internet?

+

Ja! Nach dem ersten Laden funktioniert die Demo komplett offline. Sie können Listen erstellen, bearbeiten und verwalten, auch ohne Internetverbindung.

👥 Kann ich mehrere Benutzer in der Demo testen?

+

Die Demo simuliert verschiedene Benutzerrollen. Melden Sie sich als "demo" für normale Funktionen oder als "admin" für Administratorrechte an. Jeder Account hat eigene Daten.

🔄 Wie kann ich meine Demo-Daten zurücksetzen?

+

Öffnen Sie die Browser-Entwicklertools (F12), gehen Sie zur "Application/Storage" Tab, löschen Sie die localStorage Einträge für diese Seite, und laden Sie die Seite neu.

📱 Funktioniert die Demo auf mobilen Geräten?

+

Absolut! Die App ist vollständig responsiv und funktioniert perfekt auf Smartphones und Tablets. Die Touch-Bedienung ist optimiert für mobile Nutzung.

🚀 Wie kann ich die App selbst installieren?

+

Besuchen Sie das GitHub Repository für vollständige Installationsanweisungen. Sie benötigen Node.js und können die App lokal oder auf einem Server betreiben.

🎯 Welche Browser werden unterstützt?

+

Die Demo funktioniert in allen modernen Browsern: Chrome, Firefox, Safari, Edge. Für die beste Erfahrung empfehlen wir die neueste Version Ihres bevorzugten Browsers.
